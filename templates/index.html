The Artist's Union: Get Audio

Retrieves the archived audio for a given The Artists Union track.

Retrieving the track ID

For https://theartistunion.com/tracks/900b3a the track ID would be 900b3a.

API (for use in scripts)

Call /get/X.json where X is the track ID.
